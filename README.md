# Bowler Speed MVP

React Native / Expo proof of concept for the Android Bowler Speed MVP spec.

## What is built

- Session creation and local history.
- Camera setup and pitch alignment guidance.
- Calibration form for camera height, distance behind stumps, and pitch scale.
- Camera recording screen using `expo-camera`.
- Mock speed estimation result after each ball.
- Ball replay path capture, confidence labelling, and session summary.

## Run

Install dependencies first, then start Expo:

```sh
npm install
npm run android
```

The current environment does not have `npm` available, so dependencies were not installed here.
